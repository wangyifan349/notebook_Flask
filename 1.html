🔐 安全通信软件

  • Signal — 端到端加密、开源客户端/服务器，广泛信任与使用
  • Element (Matrix) — 基于 Matrix 协议，端到端加密，完全开源
  • Tox (qTox, uTox) — 点对点加密，无中心服务器，客户端多样化
  • Jami — 分布式 P2P 网络，端到端加密，去中心化
  • Briar — Android 端 P2P/蓝牙/Tor 通信，专注离线场景
  • Wire — 端到端加密，开源客户端，企业与个人均可用
  • RiotX (Matrix) — Element 的 Android 轻量版，完全开源

🗜️ 文件压缩与备份工具

  • 7-Zip / p7zip — 高压缩比，完全开源，支持多种格式
  • PeaZip — 跨平台图形界面，基于 p7-zip/FreeArc，GPLv3
  • GNU tar — Linux/Unix 标准归档工具，与 gzip/bzip2/xz 协同
  • rsync — 增量同步、高效备份，广泛用于镜像和备份
  • Restic — 快速、安全的增量备份，AES-256 加密，完全开源
  • BorgBackup — 去重、高效加密，适合大规模数据长期存储
  • Duplicity — 支持远程存储的加密增量备份，基于 librsync

🖥 隐私操作系统与沙盒

  • Tails — 从 U 盘启动，所有流量强制走 Tor，适合临时隐私环境
  • Qubes OS — 基于 Xen 虚拟化，通过隔离不同“域”保障系统安全
  • Whonix — 双虚拟机架构，所有网络流量均经 Tor 网关
  • PureOS — 完全由自由软件基金会认定的纯开源操作系统
  • Subgraph OS — 针对抗审查与匿名通信优化的 Linux 发行版

🛡️ 网络隐私加固

  • NextDNS — 云端 DNS 过滤服务，可自定义广告、跟踪屏蔽
  • Pi-hole — 局域网级广告与跟踪拦截
  • AdGuard Home — 自托管 DNS 级广告拦截
  • DNSCrypt — 加密 DNS 查询，防止中间人劫持
  • CoreDNS — 可插件化的 DNS 服务器，支持隐私增强插件
